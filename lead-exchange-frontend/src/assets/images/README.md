# 图片资源目录

## 目录说明

此目录用于存放项目中使用的图片资源文件。

## 文件组织规范

### 目录结构

```
images/
├── logos/          # Logo图片
├── icons/          # 图标图片
├── backgrounds/    # 背景图片
├── avatars/        # 头像图片
└── common/         # 通用图片
```

### 命名规范

- 使用小写字母和连字符：`user-avatar.png`
- 包含尺寸信息：`logo-120x60.png`
- 语义化命名：`login-background.jpg`

### 格式要求

- **图标**: 优先使用SVG格式
- **照片**: 使用JPG格式
- **透明图片**: 使用PNG格式
- **动画**: 使用GIF或WebP格式

### 优化要求

- 图片大小控制在合理范围内
- 使用适当的压缩比例
- 提供多种尺寸版本（如需要）
